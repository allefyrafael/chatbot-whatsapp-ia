Conectar o banco do seu projeto

Informe os dados do banco MySQL que você criou no AWS RDS — é nele que o chatbot vai buscar e cadastrar as informações do seu projeto. Vamos testar a conexão antes de salvar.

Opções avançadas (TLS/SSL)

Onde achar esses dados

No console da AWS: RDS → Databases → clique no seu banco.

Hostname
aba Connectivity & security → campo Endpoint
Porta
mesma aba → Port (normalmente 3306)
Username
o Master username definido ao criar o banco
Password
a Master password definida ao criar o banco
Database
aba Configuration → DB name

Antes de testar, confirme no RDS: Public access = Yes e uma regra no Security Group liberando a porta 3306 para o seu IP.
