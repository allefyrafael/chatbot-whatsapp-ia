Onde achar esses dados

No console da AWS: RDS → Databases → clique no seu banco.

Hostname
aba Connectivity & security → Endpoint
Porta
mesma aba → Port (normalmente 3306)
Username
o Master username definido ao criar
Password
a Master password definida ao criar
Database
aba Configuration → DB name

Confirme no RDS: Public access = Yes e uma regra no Security Group liberando a porta 3306 para o seu IP.

Trocar para outro banco muda onde ficam os dados: empresa, administradores e produtos do banco anterior não aparecem mais, e você pode precisar refazer o cadastro inicial.
