Como pegar sua chave

  1. Acesse console.groq.com e crie uma conta gratuita.
  2. Vá em API Keys (console.groq.com/keys).
  3. Clique em Create API Key, dê um nome e confirme.
  4. Copie a chave (começa com gsk_). O Groq a mostra uma vez só.
  5. Cole ao lado e clique em Salvar e validar.

A chave é secreta. Fica guardada com segurança e nunca é exibida por completo. Se vazar, revogue no console e gere outra.
